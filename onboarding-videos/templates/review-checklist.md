# 動画レビューチェックリスト

## 台本レビュー（レビュアー: 嘉生）

### 内容の正確性
- [ ] 操作手順に誤りがない
- [ ] 用語が正しく使用されている
- [ ] 最新のUI/機能を反映している
- [ ] 説明に抜け漏れがない

### 構成・分かりやすさ
- [ ] 学習目標が明確に定義されている
- [ ] 前提知識・事前視聴が適切に設定されている
- [ ] セクション構成が論理的である
- [ ] 時間配分が適切である
- [ ] 専門用語に適切な説明がある

### 顧客視点
- [ ] 初めて使う人でも理解できる
- [ ] よくある質問(FAQ)が適切に設定されている
- [ ] 次に視聴すべき動画が案内されている

### 判定

- [ ] **承認**: 撮影に進んでOK
- [ ] **条件付き承認**: 軽微な修正後、再レビュー不要で撮影可
- [ ] **要修正**: 修正後、再レビュー必要

## 動画レビュー（レビュアー: 嘉生）

### 映像品質
- [ ] 画面が見やすい（解像度、明るさ）
- [ ] マウスカーソルの動きが分かりやすい
- [ ] 画面遷移がスムーズである
- [ ] 操作のテンポが適切である

### 音声品質
- [ ] ナレーションが聞き取りやすい
- [ ] ノイズが気にならない
- [ ] 音量が適切である（大きすぎ/小さすぎない）
- [ ] BGMの音量バランスが適切である

### テロップ・テキスト
- [ ] テロップに誤字脱字がない
- [ ] テロップの表示時間が適切である
- [ ] テロップが読みやすい（フォント、サイズ、色）
- [ ] 強調ポイントが適切にハイライトされている

### 内容の一致
- [ ] 台本通りの内容になっている
- [ ] 操作手順が正確に再現されている
- [ ] 学習目標を達成できる内容になっている

### 動画構成
- [ ] オープニング/エンディングが適切である
- [ ] 想定時間内に収まっている
- [ ] 次の動画への誘導がある

### 判定

- [ ] **公開可**: このまま公開OK
- [ ] **軽微修正**: テロップ修正など軽微な修正後、再レビュー不要で公開可
- [ ] **要修正**: 修正後、再レビュー必要
- [ ] **再撮影**: 大幅な修正が必要、再撮影
