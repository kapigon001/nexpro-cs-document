## ディレクトリ構成

```
onboarding-videos/
├── README.md           # 本ファイル
├── DESIGN.md           # 詳細設計書
├── scripts/            # 台本
│   ├── phase1/         # Phase 1（動画1-6）
│   ├── phase2/         # Phase 2（動画7-12）
│   └── phase3/         # Phase 3（動画13以降）
├── assets/             # 素材
│   ├── thumbnails/     # サムネイル
│   ├── bgm/            # BGM
│   └── images/         # 画像素材
├── exports/            # 完成動画
│   ├── phase1/
│   └── phase2/
└── templates/          # 関連テンプレート
```

## 制作フロー

```
企画 → 台本作成 → レビュー → 撮影 → 編集 → レビュー → 修正 → 公開
(1日)   (2日)     (1日)    (1日)   (2日)   (1日)    (1日)   (0.5日)
```
